# M3 Play

<p align="center">
  <img src="assets/icon.png" width="140" alt="M3 Play Icon"/>
</p>

<p align="center">Modern Android music player based on OpenTune.</p>

## Features

- Material 3 UI
- YouTube Music browsing and playback
- Lyrics support
- Offline and library features
- Android 13+ themed icon support

## App info

- App name: **M3 Play**
- Package: `com.j.m3play`

## Build

```bash
./gradlew assembleDebug
```
